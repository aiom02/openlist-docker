# 重启后端服务说明

## 问题原因
创建文件夹失败是因为数据库中还没有 `video_favorite_folder` 和 `video_favorite` 表。

## 解决方法

### 1. 停止当前运行的后端服务
按 `Ctrl + C` 停止正在运行的 OpenList 后端

### 2. 重新启动后端
运行启动脚本：
```bash
启动OpenList.bat
```
或者手动运行：
```bash
cd OpenList-main
./openlist.exe server
```

### 3. 数据库自动迁移
后端启动时会自动执行数据库迁移，创建新表：
- `video_favorite_folders` - 收藏文件夹表
- `video_favorites` - 收藏视频表

### 4. 刷新前端页面
重启后端后，刷新浏览器页面，再次尝试创建文件夹。

## 验证成功
如果看到 **"文件夹创建成功"** 的提示（没有错误提示），说明修复成功！

## 如果还有问题
检查后端日志，看是否有数据库错误信息。
