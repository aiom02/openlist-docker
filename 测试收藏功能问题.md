# 调试"添加到收藏失败"问题

## 可能的原因

### 1. Fingerprint 生成失败
由于 `buildMediaFingerprint` 是异步的，可能出错了。

### 2. Storage ID 获取问题
后端自动获取 storage_id 可能失败。

### 3. 后端验证失败
某些字段验证不通过。

## 调试步骤

### 步骤1：查看浏览器控制台
1. 按 F12 打开开发者工具
2. 切换到"Console"（控制台）标签
3. 点击"添加到收藏"按钮
4. 查看是否有错误信息

### 步骤2：查看网络请求
1. 按 F12 打开开发者工具
2. 切换到"Network"（网络）标签
3. 点击"添加到收藏"按钮
4. 找到 `/api/video_favorites/create` 请求
5. 查看：
   - Request Payload（请求参数）
   - Response（响应）
   - Status Code（状态码）

### 步骤3：查看后端日志
查看后端命令行窗口的输出，看是否有错误信息。

## 请提供以下信息

1. **浏览器控制台的错误信息**（红色的）
2. **网络请求的响应内容**
3. **后端日志的错误信息**

这样我才能准确定位问题！
