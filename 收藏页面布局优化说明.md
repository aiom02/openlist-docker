# 收藏页面布局优化说明

## 修改概述

根据用户需求，对视频收藏和音频收藏页面进行了布局优化，并在右下角工具栏添加了"我的音频收藏"快捷入口。

## 具体修改内容

### 1. 视频收藏页面布局优化

**文件**: `OpenList-Frontend-main/src/pages/video-favorites/index.tsx`

#### 修改前
- 顶部只有标题和"创建文件夹"按钮
- 布局较为简单

#### 修改后
- 顶部分为两行：
  - 第一行：返回按钮 + 标题
  - 第二行：三个操作按钮
    - **新建收藏文件夹** (蓝色主按钮)
    - **我的收藏** (边框按钮)
    - **添加到收藏** (边框按钮)

#### 按钮功能
- **新建收藏文件夹**: 打开创建文件夹对话框
- **我的收藏**: 刷新当前页面
- **添加到收藏**: 提示用户从视频播放器添加视频

### 2. 音频收藏页面布局优化

**文件**: `OpenList-Frontend-main/src/pages/audio-favorites/index.tsx`

#### 修改内容
与视频收藏页面保持完全一致的布局：
- 顶部双行布局
- 三个操作按钮：
  - **新建收藏文件夹** (蓝色主按钮)
  - **我的收藏** (边框按钮)
  - **添加到收藏** (边框按钮)

#### 按钮功能
- **新建收藏文件夹**: 打开创建文件夹对话框
- **我的收藏**: 刷新当前页面
- **添加到收藏**: 提示用户从音频播放器添加音频

### 3. 右下角工具栏添加音频收藏入口

**文件**: `OpenList-Frontend-main/src/pages/home/toolbar/Right.tsx`

#### 新增内容
在右下角的浮动工具栏中，"我的视频收藏"按钮下方添加了：
- **我的音频收藏**图标按钮 (使用音频图标)
- 点击后跳转至 `/audio-favorites` 页面
- 仅登录用户可见

#### 代码变更
```typescript
// 新增导入
import { AiOutlineAudio } from "solid-icons/ai"

// 新增按钮
<Show when={isLoggedIn()}>
  <RightIcon
    as={AiOutlineAudio}
    tips="my_audio_favorites"
    onClick={() => {
      navigate("/audio-favorites")
    }}
  />
</Show>
```

### 4. 语言文件更新

**文件**: 
- `OpenList-Frontend-main/src/lang/zh-cn/home.json`
- `OpenList-Frontend-main/src/lang/en/home.json`

#### 新增翻译
- 中文: `"my_audio_favorites": "我的音频收藏"`
- 英文: `"my_audio_favorites": "My Audio Favorites"`

## 用户体验改进

1. **统一的界面风格**: 视频收藏和音频收藏页面采用完全一致的布局
2. **快速访问**: 通过右下角工具栏可以快速访问两种收藏页面
3. **明确的操作引导**: 顶部按钮清晰地展示了主要操作选项
4. **视觉层次**: 使用颜色区分主要操作和次要操作

## 注意事项

1. **权限控制**: 音频收藏入口仅对登录用户显示
2. **操作提示**: "添加到收藏"按钮会给出友好的提示信息
3. **响应式布局**: 按钮布局在不同屏幕尺寸下保持良好显示
4. **一致性**: 两个收藏页面保持完全一致的交互逻辑

## 技术细节

### 布局结构
- 使用 `VStack` 和 `HStack` 组件构建层次化布局
- 第一行：导航元素（返回按钮 + 标题）
- 第二行：操作按钮组

### 样式设置
- 主按钮使用 `colorScheme="accent"`
- 次要按钮使用 `variant="outline"`
- 统一按钮尺寸为 `size="md"`
- 按钮间距为 `spacing="$3"`

### 图标使用
- 视频收藏: `AiOutlineHeart` (心形图标)
- 音频收藏: `AiOutlineAudio` (音频图标)
